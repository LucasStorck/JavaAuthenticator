# Sistema de Registro e Autenticação de Usuário
#### Sistema de registro e autenticação de usuários, desenvolvido com Spring Security 6, integrando autenticação baseada em JWT (JSON Web Tokens) e OAUTH 2.0.

## Tecnologias

- Java 21
- Spring
- Spring Security
- Oauth 2.0
- PostgreSQL
- Flyway
- Swagger
- Docker
